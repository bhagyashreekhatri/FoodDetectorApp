# 🍽️ Food Detector App

An iOS app that detects food from photos using Apple's CoreML and Vision framework.

## Features
- 📷 Take a photo or pick from gallery
- 🤖 On-device AI food detection (no internet needed)
- 🏆 Shows top 5 predictions with confidence scores
- ⚡ Powered by MobileNetV2 CoreML model

## Tech Stack
- SwiftUI
- Apple Vision Framework
- CoreML (MobileNetV2)

## Setup
1. Clone the repo
2. Download [MobileNetV2.mlmodel](https://ml-assets.apple.com/coreml/models/Image/ImageClassification/MobileNetV2/MobileNetV2.mlmodel)
3. Drag `MobileNetV2.mlmodel` into the Xcode project
4. Run on simulator or device (iOS 15+)
